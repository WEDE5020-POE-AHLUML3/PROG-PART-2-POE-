# Cyber Guard Assistant Chat

A WPF C# cybersecurity assistant chatbot with a modern blue themed GUI.

## Features
- Sliding side menu with saved chats
- Keyword recognition and random responses
- Sentiment detection (worried, scared, frustrated, curious)
- Memory and recall (remembers your name and favourite topic)
- Animated typing indicator (bouncing dots)
- Chat saving and loading from Documents folder
- Suggested topic buttons in side menu
- Start new chat and return to name page

## Cybersecurity Topics Covered
- Password safety
- Phishing awareness
- Malware protection
- Privacy tips
- Scam detection
- Wi-Fi security
- Two-Factor Authentication
- VPN usage

## How to Run
1. Open the solution in Visual Studio
2. Build and run the project
3. Enter your name on the welcome page
4. Start chatting with the assistant

## Technologies Used
- C# WPF (.NET Framework)
- XAML for UI design
- File I/O for chat saving and loading
- OOP principles and data structures
- Dictionary and List collections for response management

## Project Structure
- MainWindow - Landing page with animated button
- WelcomeChat - Name entry page
- CSChat - Main chat window with sliding menu

## Save Location
Chats are saved to:
`Documents\CyberGuardChats\`
